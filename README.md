# SmartPharm - Smart Hardware Solutions

A modern, minimalistic B2B landing page for SmartPharm, showcasing smart hardware solutions with integrated analytics software.

## Features

- **Modern React Framework**: Built with React 18 and Vite for optimal performance
- **Tailwind CSS**: Minimalistic, modern UI design
- **Responsive Design**: Fully responsive across all devices
- **B2B Focused**: Enterprise-grade messaging and design

## Products Showcased

- Electronic Shelf Labels (ESLs)
- Digital Posters
- Price Check Kiosks
- Kiosk Screens
- Backless Labels

## Tech Stack

- React 18
- Vite
- Tailwind CSS
- Lucide React (Icons)
- Framer Motion (Animations)

## Getting Started

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

The application will start on `http://localhost:3000`

### Build

```bash
npm run build
```

### Preview Production Build

```bash
npm run preview
```
